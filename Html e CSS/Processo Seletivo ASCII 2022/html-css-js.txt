A referência ao script do javascript pode ser feita em qualquer lugar do arquivo html. Mas é bom colocar logo antes de onde vai ser usado para deixar o código mais legível e entendível.
É possível definir css (tag <style>) e javascript diretono arquivo html, mas isso não é nada recomendado. 
O recomendado é escrever um arquivo css e um arquivo javascript e simplesmente incluí-los no arquivo html com as devidas tags:
	<link> para css
	
O body já vem com um pré-valor de paddng, o qual não é zero
Se não tirar esse valor default do body e de outras tags talvez não seja possível deixar elas coladas)

O * é para indicar todas as tags(CSS)


position: relative --> deixa o container se mover
	left e top movem o container na quantidade de pixels
